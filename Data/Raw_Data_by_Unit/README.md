Data Dictionary: Specific Capacity and Hydrogeologic Units
Folder: Data/Raw_Data_by_Unit/

This folder contains the primary field measurements used to validate the universal power-law formula for effective porosity (ne​).
Column Header	Description	Units
ID	Unique identifier for each well record (1–609).	Integer
Yield_m3_day	Pumping rate or discharge during the well test.	m3/day
Drawdown_m	Total drop in water level measured in the well.	Meters (m)
Spec_Cap_m2_day	Specific Capacity (q/s), calculated as Yield / Drawdown.	m2/day
Lithology	Primary geologic unit identified from driller's logs.	Code (see below)
